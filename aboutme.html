Some Projects I Worked On!!

Midnight Pizza Run

The first big project I worked on with my group was a game called Midnight Pizza Run and our group made it using Scratch. I think it was a good idea making our first project using Scratch because the building blocks made it super easy to compile things together and it also gave me a godo idea of the basics of computer science in a fun way. The hardest part of the game was making the sprite to jump higher when the up button is pressed. Also, it was a bit challenging making sure that the pizza ingredients don't overlap on each other. Overall, it was really fun working with my group and coming up with the idea of the game and I was satisfied with the end project we got.

Shapes!!!

After completing our Scratch project, we moved on to learn Python. In this project, we used the Turtle library to complete our project. The only challenging part was making sure the angles are correct in order for the turtle to make it draw a certain shape. It was also fun filling in the shapes with different colors and making the icon into a little turtle!

PEEP

For this Python project, we worked on obamifying Pepe and it was super fun because we took a image and changed the colors based on the original pixels. Text Adventure

For this Python Project, we worked on making a text adventure game. This game took the user input and based on that, the game will give a specific response. It was fun working with my group to come up with a story for this adventure.
